# OpenClaw Stage II Attack Test Page

> ⚠️ **Security Research Only**: This repository contains materials for academic research on LLM Agent security vulnerabilities. For authorized security testing and educational purposes only.

## Purpose

This GitHub Pages site is used to test **Indirect Prompt Injection (IPI)** attacks against OpenClaw Agent as part of academic research on:
- LLM Agent safety and security
- Multi-stage attack chains in autonomous AI systems
- Defense mechanisms against prompt injection

## Research Context

This work is part of a security research study on autonomous LLM agents, following the threat model described in academic literature on AI safety.
